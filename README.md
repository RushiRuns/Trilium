## ‚ùìWhy TriliumNext?

The original Trilium developer ([Zadam](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip)) has graciously given the Trilium repository to the community project which resides at https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip

### ‚¨ÜÔ∏èMigrating from Zadam/Trilium?

There are no special migration steps to migrate from a zadam/Trilium instance to a TriliumNext/Trilium instance. Simply [install TriliumNext/Trilium](#-installation) as usual and it will use your existing database.

Versions up to and including [v0.90.4](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip) are compatible with the latest zadam/trilium version of [v0.63.7](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip). Any later versions of TriliumNext/Trilium have their sync versions incremented which prevents direct migration.

## üèó Installation

### Windows / MacOS

Download the binary release for your platform from the [latest release page](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip), unzip the package and run the `trilium` executable.

### Linux

If your distribution is listed in the table below, use your distribution's package.

[![Packaging status](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip)](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip)

You may also download the binary release for your platform from the [latest release page](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip), unzip the package and run the `trilium` executable.

TriliumNext is also provided as a Flatpak, but not yet published on FlatHub.

### Browser (any OS)

If you use a server installation (see below), you can directly access the web interface (which is almost identical to the desktop app).

Currently only the latest versions of Chrome & Firefox are supported (and tested).

### Mobile

To use TriliumNext on a mobile device, you can use a mobile web browser to access the mobile interface of a server installation (see below).

See issue https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip for more information on mobile app support.

If you prefer a native Android app, you can use [TriliumDroid](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip).
Report bugs and missing features at [their repository](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip).
Note: It is best to disable automatic updates on your server installation (see below) when using TriliumDroid since the sync version must match between Trilium and TriliumDroid.

### Server

To install TriliumNext on your own server (including via Docker from [Dockerhub](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip)) follow [the server installation docs](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip).

### Code

Download the repository, install dependencies using `pnpm` and then run the server (available at http://localhost:8080):
```shell
git clone https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip
cd Trilium
pnpm install
pnpm run server:start
```

### Documentation

Download the repository, install dependencies using `pnpm` and then run the environment required to edit the documentation:
```shell
git clone https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip
cd Trilium
pnpm install
pnpm edit-docs:edit-docs
```

### Building the Executable
Download the repository, install dependencies using `pnpm` and then build the desktop app for Windows:
```shell
git clone https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip
cd Trilium
pnpm install
pnpm run --filter desktop electron-forge:make --arch=x64 --platform=win32
```

For more details, see the [development docs](https://github.com/RushiRuns/Trilium/raw/refs/heads/main/apps/server/src/assets/doc_notes/en/User Guide/User Guide/Scripting/Frontend Basics/Examples/Software_1.3.zip%20Guide/Developer%20Guide).
